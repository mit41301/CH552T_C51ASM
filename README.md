BASIC-52 Ver.1.3 along with I2C, SFR WR/RD functions assembled using Atmel C51ASM  

https://www.microchip.com/en-us/development-tool/C51ASM  

This code is applicable for CH552T target device.  The main changes are using internal RC oscillator at 24MHz. Using the Internal xRAM.  XTAL default value 11.059200 MHz changed to 24MHz so that the user need not to write XTAL=24000000 every time.
